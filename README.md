# 🎂 Responsive Javascript Age Calculator App

A clean and responsive **Age Calculator Web Application** built using **HTML, CSS, and JavaScript**.  
This application allows users to select their **date of birth** and instantly calculates their **exact age in years, months, and days**.

The project focuses on practicing **JavaScript Date API**, **DOM manipulation**, and **responsive web design**.

# ✨ Features

- 📅 Select date of birth using a calendar picker
- ⚡ Instant age calculation
- 🧠 Calculates **years, months, and days**
- 📱 Fully responsive design
- 🎨 Clean and simple UI
- 🚫 Prevents selecting future dates
- 🖥 Works on all modern browsers

---

# 🛠 Technologies Used

- **HTML5**  
  Used to structure the web application.

- **CSS3**  
  Used for styling, layout, and responsive design.

- **JavaScript (ES6)**  
  Used to calculate age and handle user interaction.

---

# ⚙️ How It Works

1. The user selects their **date of birth** from the date input field.
2. JavaScript reads the selected date.
3. The current date is fetched using the **JavaScript Date object**.
4. The program calculates the difference between the **current date** and **birth date**.
5. The calculated **years, months, and days** are displayed on the screen.
